# 분리 학습 실험 (폐기)

## 실험 개요
전남(전라남도)이 가중 MAE의 77%를 차지하는 구조적 문제 해결을 위해
전남 단독 모델 + 16개 지역 통합 모델로 분리 학습 시도.

## 실험 결정 배경
- 인수인계 문서에서 "전남이 가중 MAE의 77~78%를 차지" 분석
- 분포 shift 진단(16/17개 지역 KS p<0.01)에서 capacity 증가 확인
- "통합 모델이 전남 때문에 다른 지역 학습을 방해하지 않나" 가설 검증 필요

## 실험 설계
- 모델 1: 전남 단독 XGBoost (전남 데이터만)
- 모델 2: 16개 통합 XGBoost (전남 제외)
- 학습 조건: 통합 모델(national_xgboost_model.json)과 완전 동일
  (피처 24개, 동일 하이퍼파라미터, log1p·연도 가중치 없음)
- 평가: 3개 메인 ESS 지표 (전남/16개 평균/가중 평균) + 참고용 전국 통합

## 핵심 결과

### MAE 지표 (악화)
| 지표 | 통합 (baseline) | 분리 학습 | 변화율 |
|---|---|---|---|
| 단순 평균 MAE | 9.59 | 10.66 | +11.2% 악화 |
| 가중 MAE | 31.91 | 34.44 | +7.9% 악화 |

### ESS 점수 (변화 없음)
| 지표 | 통합 (baseline) | 분리 학습 | 변화율 |
|---|---|---|---|
| 전남 ESS | 24.60 | 24.50 | -0.4% |
| 16개 평균 ESS | 18.53 | 18.49 | -0.2% |
| 가중 평균 ESS | 28.67 | 28.63 | -0.1% |
| 충남 ESS | 25.94 | 25.94 | 0.0% |

### 지역별 MAE 변화 패턴
- 소도시(서울·대전·부산·광주 등): 일관된 개선 (+10~25%)
- 대규모 태양광 지역(충남·전북·경북): 큰 폭 악화 (-20~53%)
- 충남 MAE: 17.22 → 26.39 (+53% 악화)

## 폐기 결정 사유

1. **3가지 메인 ESS 지표 모두 0%대 변화** — 통계적으로 0과 구분 불가
2. **MAE는 11% 악화** — 분리 학습이 명백히 손해
3. **소도시 개선과 대규모 지역 악화가 상쇄** — 가중 평균 차원에서 효과 무효화

## 핵심 발견 (실험의 가치)

### 발견 1: MAE와 ESS 점수는 다른 지표
충남 MAE +53% 악화에도 ESS 점수 0.00 변화 →
ESS 점수는 방향성 예측(충방전 결정)이 절대 오차보다 중요함을 실증.
"오차 지표(MAE)와 운영 효율 지표(ESS)는 다른 차원이며 둘 다 봐야 한다"

### 발견 2: 전남의 역할은 양면적
통합 모델에서 전남은 두 가지 역할을 동시에 하고 있었음:
- (-) 소도시 학습 방해 (분리 후 개선)
- (+) 대규모 지역 학습 도움 (분리 후 악화)
종합 ESS 점수에서는 두 효과가 상쇄됨.

### 발견 3: 다음 개선 방향
현재 XGBoost 통합 모델은 합리적 선택이며,
더 큰 개선은 다른 차원에서 와야 함:
- 모델 아키텍처 변경 (트랜스포머)
- 피처 엔지니어링
- 정규화 전략
