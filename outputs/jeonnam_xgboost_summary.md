# 전남 XGBoost 단독 모델 — XGBoost 챕터 마무리

## 진단 요약

- 통합 XGBoost에서 전남 MAE 90.42 (전국 평균을 9배 끌어올리는 이상치)
- 원인 1: 외삽 — 학습 max 999 MWh, 테스트 max 1,824 MWh (1.8배 초과)
- 원인 2: 설비 급증 — 2022년부터 capacity 폭증, 최근 데이터 패턴이 학습 분포와 괴리

## 시도한 방법과 결과

| 시도 | 전남 MAE | 변화 |
|---|---|---|
| 통합 모델 (baseline) | 90.42 | — |
| 1. 단독 모델 분리만 | 91.01 | -0.7% (악화) |
| 2. + log1p 타겟 변환 + 연도별 sample_weight | 86.68 | **+4.1%** |

**가장 큰 성과**: 피크 MAE 83.86 → 63.00 (-24.9%)
ESS 운영 핵심 시간대(10~14시)에서 의미 있는 개선

## 도달한 한계

- RMSE 거의 변화 없음 (190.93 → 190.91): 극단적 이상치는 여전히 못 맞힘
- log 공간 외삽 격차 0.60 잔존 (train max 6.91 vs test max 7.51)
- best_iteration=499/500 (early stopping 미발동) — validation이 test 분포 대표 못함 가능성
- 전남 ESS 점수 소폭 하락 (24.6 → 24.2): MAE 개선이 ESS 충전 타이밍 정확도로 직결되지 않음

## 다음 단계로 넘기는 이유

capacity 정규화(power/capacity 비율 예측)는 이론적으로 외삽 문제를 직접 해결할 수 있으나,
실제 capacity 데이터가 없어 proxy 추정에 의존해야 함. 본 프로젝트의 "근거 있는 수치 + 진단 기반" 원칙과 맞지 않음.
대신 트랜스포머 모델에서 다른 접근(지역별 정규화, attention 기반 패턴 학습, multi-task)으로 우회 시도.

## 하이브리드 결과 (전남 단독 모델 적용 후 전국)

| 항목 | 통합 모델 | 하이브리드 | 변화 |
|---|---|---|---|
| 전국 MAE | 9.66 | **9.42** | **+2.5%** |
| 전국 피크 MAE | — | 6.40 | — |
| 전남 MAE | 90.42 | 86.68 | +4.1% |
| ESS 운영효율점수 (전국) | 12.69 | 12.69 | 0.0% |
| ESS 점수 (전남) | 24.6 | 24.2 | -0.4pt |

**해석**: 전국 MAE는 2.5% 개선됐으나, ESS 운영효율점수는 변화 없음.
전남 1개 지역의 MAE 개선이 17개 지역 합산 예측 타이밍에 영향을 주기에는 규모가 작고,
피크 MAE 개선이 ESS 충전 결정 정확도 개선으로 직접 이어지지는 않았음.
XGBoost 한계 확인 — 트랜스포머 모델에서 지역별 패턴 학습으로 재시도 예정.
